# EnglishTracker

英語学習の記録とモチベーション管理をサポートするiOSアプリです。

## 概要

EnglishTrackerは、英語学習の進捗を記録し、連続学習ストリークとポイントシステムで学習のモチベーションを維持することを目的としたアプリです。

## 主な機能

### 1. 学習記録機能
- 日付ごとに学習時間を記録（分単位）
- 学習内容のメモ（任意）
- 学習した単語数や問題数の記録

### 2. 連続学習ストリーク機能
- 連続学習日数を自動カウント
- ストリーク継続中は炎アイコンで視覚的に強調表示
- 1日でも途切れるとリセット

### 3. ポイント・報酬システム
- **基本**: 学習1分 = 1ポイント
- **ストリークボーナス**:
  - 3日連続: 1.5倍
  - 7日連続: 2倍
  - 14日連続: 3倍
  - 30日連続: 5倍
- 累計ポイント表示

### 4. UI構成

#### ホーム画面
- 今日の学習時間入力
- 現在のストリーク日数（大きく表示）
- 今日獲得したポイント
- ボーナス倍率の表示

#### 履歴画面
- カレンダービューで学習日を可視化
- 過去の学習記録一覧

#### 統計画面
- 累計学習時間
- 累計ポイント
- 最長ストリーク記録
- 達成バッジ

## ビルド手順

1. **プロジェクトを開く**
   ```bash
   open EnglishTracker.xcodeproj
   ```

2. **シミュレーターまたは実機を選択**
   - Xcodeの上部メニューから対象デバイスを選択

3. **ビルドして実行**
   - `Command + R` を押してビルド＆実行
   - または、Xcodeの「Run」ボタンをクリック

## 機能詳細

### ストリーク計算ロジック
- 毎日の学習記録から連続日数を自動計算
- 日付が1日でも途切れるとストリークはリセット
- 最長ストリーク記録も保存・表示

### ポイントシステム
- 学習時間に基づいてポイントを付与
- ストリーク日数に応じてボーナス倍率が適用
- 累計ポイントで学習の総量を可視化

### データ永続化
- Swift Dataを使用してローカルに保存
- アプリ終了後もデータが保持される
- 初回起動時はクリーンな状態
