# 🎹 PianoProgress

> ピアノ演奏の練習を効率的に管理・追跡するiOSアプリケーション

## 📖 プロジェクト概要

PianoProgressは、ピアノ学習者が日々の練習を記録し、進捗を可視化し、効率的な練習をサポートするためのiOSアプリケーションです。メトロノーム機能、練習時間の追跡、統計分析など、ピアノ練習に必要な機能を一つのアプリに統合しています。

## ✨ 主要機能

### 🎯 練習管理
- **練習セッション記録**: ワンタップで練習の開始・終了を記録
- **タイマー機能**: 集中して練習できるタイマー
- **曲リスト管理**: 練習中の曲を整理・管理
- **練習メモ**: セッションごとに気づきを記録

### 📊 進捗追跡
- **カレンダービュー**: 練習日を視覚的に確認
- **統計グラフ**: 週間・月間の練習時間を可視化
- **目標設定**: 日別・週別の練習目標を設定
- **ストリーク表示**: 連続練習日数でモチベーション維持

### 🎼 練習サポート
- **メトロノーム**: BPM調整、拍子設定、アクセント機能
- **音楽理論参照**: スケール、コードの基本情報
- **リマインダー**: 練習時間の通知

## 📱 対応環境

- **対応OS**: iOS 17.0以上
- **対応デバイス**: iPhone
- **言語**: Swift 5.9+
- **フレームワーク**: SwiftUI, SwiftData

## 🏗️ アーキテクチャ

本アプリは以下のアーキテクチャパターンを採用しています：

- **MVVM (Model-View-ViewModel)**: UIとビジネスロジックの分離
- **Repository パターン**: データアクセス層の抽象化
- **依存性注入**: テスタビリティと保守性の向上

詳細は [ARCHITECTURE.md](ARCHITECTURE.md) を参照してください。

## 📂 プロジェクト構成

```
PianoProgress/

├── Repositories/          # データアクセス層
├── Services/              # サービス層
└── Resources/             # リソース（アセット、サウンド等）
```

## 📦 使用技術・フレームワーク

### Appleフレームワーク
- **SwiftUI**: 宣言的UI構築
- **SwiftData**: データ永続化
- **AVFoundation**: メトロノーム音源
- **Charts**: グラフ表示 (iOS 16+)
- **UserNotifications**: 通知機能

### サードパーティライブラリ
現在のところ、外部ライブラリへの依存はありません。Apple標準フレームワークのみを使用し、アプリサイズの最小化とメンテナンス性の向上を図っています。

## 🔒 プライバシー・セキュリティ

- **ローカルファースト**: すべてのデータはデバイス内に保存
- **外部送信なし**: ユーザーデータを外部サーバーに送信しません
- **透明性**: データの使用目的を明示
